Implementación de VPN IPsec Site-to-Site

Interconexión Segura de Sucursales

Ingeniero: Aaron Efraim Mata Martinez | Matrícula: 179419
Plataforma: Cisco Packet Tracer (Router 1941) | Fecha: 16/02/2026

1. Escenario y Topología

El objetivo es establecer un túnel VPN seguro a través de una red pública (Internet) para conectar dos redes privadas, garantizando la confidencialidad, integridad y autenticación de los datos mediante el protocolo IPsec.

Tabla de Direccionamiento

Dispositivo Interfaz Dirección IP Máscara Rol
R1 G0/1 (LAN) 192.168.1.1 255.255.255.0 Gateway Local
R1 S0/0/0 (WAN) 10.1.1.1 255.255.255.252 Extremo VPN
R2 G0/1 (LAN) 192.168.3.1 255.255.255.0 Gateway Remoto
R2 S0/0/0 (WAN) 10.2.2.2 255.255.255.252 Extremo VPN

2. Preparación del Router (Licenciamiento)

Los routers Cisco serie 1900 requieren la activación del paquete de seguridad para habilitar los comandos criptográficos.

Router> enable
Router# license boot module c1900 technology-package securityk9
Router# copy running-config startup-config
Router# reload
Nota: Sin este paso, los comandos "crypto" no estarán disponibles.

3. Configuración R1

Paso A: Tráfico Interesante (ACL)
ip access-list extended VPN-ACL
permit ip 192.168.1.0 0.0.0.255 192.168.3.0 0.0.0.255
Paso B: Fase 1 (ISAKMP)
crypto isakmp policy 10
encryption aes 256
hash sha256
authentication pre-share
group 5
exit
crypto isakmp key cisco123 address 10.2.2.2
Paso C: Ruta Estática
ip route 0.0.0.0 0.0.0.0 10.1.1.2

4. Configuración R2

Paso A: Tráfico Interesante (Espejo)
ip access-list extended VPN-ACL
permit ip 192.168.3.0 0.0.0.255 192.168.1.0 0.0.0.255
Paso B: Fase 1 (ISAKMP)
crypto isakmp policy 10
encryption aes 256
hash sha256
authentication pre-share
group 5
exit
crypto isakmp key cisco123 address 10.1.1.1
Paso C: Ruta Estática
ip route 0.0.0.0 0.0.0.0 10.2.2.1

5. Fase 2 y Crypto Map (Ambos Routers)

Configuración del Transform Set para proteger los datos y aplicación a la interfaz WAN.

! Definir Transform Set
crypto ipsec transform-set MYSET esp-aes 256 esp-sha256-hmac

! Crear Crypto Map
crypto map MYMAP 10 ipsec-isakmp
set peer [IP_DEL_REMOTO]
set transform-set MYSET
match address VPN-ACL

! Aplicar a la Interfaz
interface Serial0/0/0
crypto map MYMAP

6. Verificación y Troubleshooting

Comandos esenciales para validar el estado del túnel una vez generado tráfico (Ping).

Comando Resultado Esperado
show crypto isakmp sa Estado QM_IDLE (Indica Fase 1 exitosa).
show crypto ipsec sa Contadores de paquetes (pkts encaps/decaps) incrementando.
show crypto map Muestra la asociación entre la ACL, el Peer y el Transform Set.
